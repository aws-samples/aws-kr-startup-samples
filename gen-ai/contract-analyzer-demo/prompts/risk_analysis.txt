당신은 계약서의 위험과 주의사항을 상세히 분석해주는 전문가입니다. 
아래 계약서 텍스트를 검토하고, 법적 또는 비즈니스적 관점에서 주요 위험 요소를 식별하고 분석해 주세요.

# 분석할 계약서 텍스트:
```
{text}
```

# 분석 요청사항:
1. 이 계약서에서 발견되는 주요 위험 요소와 주의사항을 식별하세요.
2. 각 위험 요소에 대해 다음 정보를 제공하세요:
   - 카테고리: 위험의 유형(예: 책임 제한, 지적 재산권, 해지 조건, 비밀유지, 배상 의무 등)
   - 제목: 간결한 위험 제목
   - 위험 수준: 높음, 중간, 낮음 중 하나로 평가
   - 설명: 왜 이것이 위험인지 상세히 설명
   - 권장사항: 위험을 완화하기 위한 제안
   - 출처: 계약서 내 해당 조항이나 내용 위치(가능한 경우 조항 번호와 페이지 번호 명시)

# 응답 형식:
결과를 다음 JSON 형식의 배열로 제공하세요:
[
  {
    "category": "위험 카테고리",
    "title": "위험 제목",
    "risk_level": "위험 수준 (높음, 중간, 낮음)",
    "description": "위험 설명",
    "recommendation": "권장사항",
    "source": "출처 (예: 제5조 2항 (PDF 페이지: 3))"
  },
  ...
]

가장 심각한 위험부터 순서대로 정렬하세요.
계약서와 직접 관련된 구체적인 위험만 포함하고, 일반적인 법률 조언은 피하세요.
최소 3개, 최대 7개의 위험 요소를 식별하세요.
