# AI 모델 프롬프트 템플릿

이 디렉토리는 계약서 분석 앱이 사용하는 AI 모델 프롬프트 템플릿 파일을 포함합니다. 이 템플릿 파일은 애플리케이션이 실행될 때 동적으로 로드되므로, 앱을 재배포하거나 코드를 수정하지 않고도 프롬프트를 사용자화할 수 있습니다.

## 현재 사용 가능한 템플릿

- `summarization.txt`: 계약서 요약 프롬프트
- `extraction.txt`: 계약서 필드 추출 프롬프트
- `risk_analysis.txt`: 계약서 위험 분석 프롬프트

## 프롬프트 수정 방법

1. 해당 템플릿 파일을 텍스트 편집기로 열어 수정합니다.
2. 프롬프트 내의 `{text}` 플레이스홀더는 그대로 유지해야 합니다. 이 부분은 실제 계약서 텍스트로 대체됩니다.
3. 프롬프트의 지시사항과 형식을 필요에 따라 수정할 수 있습니다.
4. 파일을 저장합니다. 변경 사항은 다음 API 호출부터 바로 적용됩니다.

## 프롬프트 파일 형식

각 프롬프트 파일은 일반 텍스트 형식이며, 다음과 같은 구조를 따릅니다:

1. 초기 지시사항: AI 모델에게 수행해야 할 작업 설명
2. 분석 대상 텍스트 표시: `{text}` 플레이스홀더 포함
3. 출력 형식 지정: JSON 형식 또는 기타 구조화된 형식으로 응답하도록 요청

## 주의사항

- 프롬프트를 수정할 때는 출력 형식을 변경하지 않는 것이 좋습니다. 응답 형식이 바뀌면 애플리케이션이 결과를 처리하지 못할 수 있습니다.
- AI 모델에게 너무 복잡한 지시를 내리면 응답 품질이 떨어질 수 있습니다.
- 프롬프트 파일이 없거나 읽을 수 없는 경우, 앱은 오류 메시지를 표시합니다.
